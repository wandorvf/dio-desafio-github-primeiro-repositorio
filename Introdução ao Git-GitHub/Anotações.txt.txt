## Comandos Básicos do GIT Bash

cd = entrar na pasta
dir = listar
mkdir = criar pasta
del/rmdir = deletar pasta


Ctrl + l = Limpar tela

echo > = Criar arquivo

## Comandos básicos de GIT

- Iniciar GIT = git init
- Iniciar Versionamento = git add
- Criar um Commit = git commit



### Empurrar código do diretório da máquina para o GitHub

1 - Adicionar o diretório = git add

2 - criar um commit para o diretório = git commit -m "Diretório"

3 - empurrar arquivo p/ github = git push origin Master or Main


 - Clonar um diretório do GitHub para a máquina = git clone + link https do GitHub
